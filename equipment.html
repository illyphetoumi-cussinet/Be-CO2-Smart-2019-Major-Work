5.b. 2D GRAPHICS DESIGN and ANIMATION SOFTWARE APPLICATION PROGRAMS SELECTED

 

Software Program

DESCRIPTION

JUSTIFICATION and SELECTION

Adobe Illustrator 2019

Adobe Illustrator is a vector graphics editor.

POSITIVES:

  • Scalable and Resolution Independent
  • Helpful user interface
  • It works on any computer system.

NEGATIVES:

  • Requires a lot of space on the display in order to use the program properly.
  • It is a learning curve for beginners.
  • Interface is very similar to Adobe Photoshop.

SELECTION:

  • This is the software application I will select to design my global warming animations and statistics, because it is useful especially for creating Scalable and Resolution Independent vector graphics or images.


    • Adobe Photoshop CC 2019

    Adobe Photoshop is an image editing program.

    POSITIVES:

    • Very easy to make small changes while editing.
    • Able to strengthen editing skills.
    • Able to take standard photos to a professional level.

    NEGATIVES:

    • This program is a good learning resource, but it is not well suited for beginners, since it has very basic functions that beginners can easily find out on his or her own.
    • It is a very expensive program.
    • Some of the advanced functions in this program are nowhere to be found.
    • Whenever you use Photoshop for resizing a vector graphic, the result will look distorted when you export the image.

    SELECTION:

    • With great modes and editing tools, Photoshop will be selected for the design of the logo as it is very useful in 2D Graphics.

    Adobe After Effects 2019

    Adobe After Effects is a visual effects and motion graphics application. It can support the animation of 2D graphics

    POSITIVES:

    • Adobe After effects has a vast variety of animation possibilities

    NEGATIVES:

    • Not so easy to use for beginners but I am used to it

    SELECTION:

    • This is the program I will use as I am quite proficient with its use and it is a performant application for animation.

    Adobe Character Animator

    Adobe Character Animator is a great animation tool for animating characters

    POSITIVES:

    • This application uses facial recondition

    NEGATIVES:

    • It can only be used for actual characters not for all 2D graphics.

    SELECTION:

    • For my robot I would need some character animation and I will select this program.

    ADOBE ANIMATE

    Adobe Animate is a multimedia program used to animate graphics.

    It is the old Flash.

    POSITIVES:

    • It is an application commonly used and it saves a lot of time compared to other applications

    NEGATIVES:

    • I haven’t used it for 3 or 4 years now so I will need to get into it again and it is not so easy to use at first.

    SELECTION:

    • I will not use Adobe Animate but Adobe Character Animator.

    SOFTWARE PROGRAM

    DESCRIPTION

    JUSTIFICATION and SELECTION

    This application is a timeline-based video editing app which allows editing of videos.

    POSITIVES:

    • Flexible trimming tools

    • Good image stabilization and plug-ins, effects and presets.

    NEGATIVES:

    • Can get overloaded and crash then

    it needs to be restarted

    • Slower rendering than Final Cut Pro

    SELECTION:

    • I will be selecting Adobe Premiere Pro as I am very experienced in using it and it will integrate well with the other products of the Adobe Creative Cloud Suite I am using.

    NEGATIVES:

    • Not good interface to the Adobe Creative Cloud Suite products
    • Not many video effects.

    • I will not be using it as I will not be using a Mac computer and this program does not work with the Adobe suite.    .
